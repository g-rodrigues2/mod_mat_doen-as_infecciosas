Modelagem Matemática de Doenças Infecciosas (Estudos)¶

Atividade (18/03/24): Modelagem envolvendo doenças sexualmente transmissíveis¶

Hipóteses para a modelagem

  • Existem dois grupos, um de alto risco ,$N_H$, e outro de baixo risco, $N_L$.
  • A dinâmica da doença é do tipo $SIS$ com dinâmica vital.
  • Não há migração entre os grupos, indivíduos que nascem em um grupo passam toda a vida nele. Portanto, $N_H$ e $N_L$ são constantes, i.e., $N_H=S_H+I_H$ e $N_L=S_L+I_L$.
  • População total $N_0=N_H+N_L$.

O modelo matemático, segundo Keeling e Rohani (2008), é dado por:

$$\left\{\begin{array}{llll} \dfrac{dI_H}{dt}=\beta_{HH} S_H I_H+\beta_{HL}S_HI_L-\gamma I_H,\\ \\ \dfrac{dI_L}{dt}=\beta_{LH} S_LI_H - \beta_{LL}S_LI_L-\gamma I_L. \end{array}\right.$$

Reescrevendo o modelo incorporando as hipóteses anteriormente mencionadas, temos,

$$\left\{\begin{array}{llll} \dfrac{di_H}{dt}=\beta_{HH}N_0 s_H i_H+\beta_{HL}N_0 s_H i_L-(\eta+\mu) i_H,\\ \\ \dfrac{di_L}{dt}=\beta_{LH}N_0s_Li_H + \beta_{LL}N_0s_Li_L-(\eta+\mu) i_L, \end{array}\right.$$

em que $s_H=\dfrac{S_H}{N_0}$, $i_H=\dfrac{I_H}{N_0}$, $s_L=\dfrac{S_L}{N_0}$ e $i_L=\dfrac{I_L}{N_0}$. Em $t=0$, temos que $s_H(0)\approx n_H=\dfrac{N_H}{N_H+N_L}$ e $s_L(0)\approx n_L=\dfrac{N_L}{N_H+N_L}$ e, além disso, $n_H=i_H+s_H$ e $n_L=i_L+s_L$. Considerando $n_H=0,2$ e $n_L=0,8$, o modelo pode ser reescrito como

$$\left\{\begin{array}{llll} \dfrac{di_H}{dt}=\beta_{HH}N_0 (0,2-i_H) i_H+\beta_{HL}N_0 (0,2-i_H) i_L-(\eta+\mu) i_H,\\ \\ \dfrac{di_L}{dt}=\beta_{LH}N_0(0,8-i_L)i_H + \beta_{LL}N_0(0,8-i_L)i_L-(\eta+\mu) i_L, \end{array}\right.$$

A matriz de transmissão ($\beta_{H\leftarrow L}$) é dada por

$$\bar{\beta}=\begin{pmatrix} \beta_{HH}N_0 & \beta_{HL}N_0\\ \\ \beta_{LH}N_0 & \beta_{LL}N_0 \end{pmatrix}.$$

Keeling e Rohani (2008) conceitualizam matrizes de transmissão, frequentemente referidas como matrizes WAIFW (Who Acquires Infection From Whom), como ferramentas para capturar a interação entre diferentes estratos sociais. A matriz $\hat\beta$ é analogamente relevante ao parâmetro escalar $\beta$ em modelos não estruturados. Esta análise visa esclarecer essa relação. Inicialmente, avaliamos a importância relativa dos termos da matriz. É esperado que o risco de infecção seja maior para os indivíduos em grupos de alto risco, sugerindo que a soma dos parâmetros de transmissão intra e intergrupos ($\beta_{HH} + \beta_{HL}$) seja maior que a soma dos parâmetros entre grupos de baixo risco ($\beta_{LH} + \beta_{LL}$). Além disso, é presumível uma associação seletiva, onde indivíduos de alto risco têm maior probabilidade de interagir entre si, enquanto aqueles de baixo risco tendem a ter relacionamentos com outros membros do mesmo estrato. Esta tendência sugere que os elementos diagonais da matriz são predominantes, sendo $\beta_{HH}$ o mais significativo. Por fim, assumimos interações simétricas entre grupos, implicando que o número de interações entre grupos de alto e baixo risco seja igual ao de baixo para alto; isso implica $\beta_{HL} = \beta_{LH}$, ou, mais amplamente, que a matriz seja simétrica. Essa suposição reflete a igualdade na resposta dos indivíduos de ambos os estratos ao desafio infeccioso; no entanto, se um grupo for mais suscetível por algum motivo, a simetria pode ser comprometida.

Supondo que existam dois limiares, um para cada classe, de maneira que

$$R_0^H=\dfrac{\beta_{HH} N_0 s_H}{\eta+\mu}+\dfrac{\beta_{LH} N_0s_L}{\eta+\mu} $$$$R_0^L=\dfrac{\beta_{LL} N_0 s_L}{\eta+\mu}+\dfrac{\beta_{HL} N_0s_H}{\eta+\mu} $$

Logo, pretende-se analisar por meio dos casos a seguir, a relação dos autovalores e autovetores associados a matriz de transmissão $\bar{\beta}$ com o número reprodutivo básico $R_0$ em cada caso. Espera-se que a relação ocorra entre o maior autovalor associado a matriz, o coeficiente angular associado a reta formada no crescimento da população de indivíduos infectados.

Seja $n_H=0.2$, $n_L=0.8$, $\eta = 1$ e $\mu=0$. Supondo que $\beta_{HH}N_0=10$, $\beta_{HL}N_0=\beta_{LH}N_0=0.1$ e $\beta_{LL}N_0=1$.

import numpy as np
from scipy.integrate import solve_ivp
import pandas as pd

pd.set_option('display.max_columns', None)
pd.set_option('display.max_rows', None)

def modelo(t, y, beta_HH_N0, beta_HL_N0, beta_LH_N0, beta_LL_N0, eta, mu):

    i_H, i_L = y
    
    dydt = [
        (beta_HH_N0 * (0.2 - i_H) * i_H) + beta_HL_N0 * (0.2 - i_H) * i_L - (eta + mu) * i_H,
        (beta_LH_N0 * (0.8 - i_L) * i_H) + beta_LL_N0 * (0.8 - i_L) * i_L - (eta + mu) * i_L
    ]
    
    return dydt

def R0(beta_HH, beta_HL, beta_LH, beta_LL, eta, mu, nH, nL):

    R0_H = (beta_HH_N0 * nH + beta_LH_N0 * nL) / (eta + mu)
    R0_L = (beta_LL_N0 * nL + beta_HL_N0 * nH) / (eta + mu)
    
    return R0_H, R0_L

def resolver_sistema(a, b, c, d):
    # Verifica se o sistema é possível de resolver
    if a == c and b == d:
        print("O sistema tem infinitas soluções.")
        return None
    elif a - c == 0:
        print("O sistema não tem solução.")
        return None
    
    # Resolve para x
    x = (d - b) / (a - c)
    
     # Calcula o valor de iL usando x na expressão iL*x + iL = 1
    iL = 1 / (x + 1)
    
    iH = 1 - iL
    
    return x, iL, iH

nH=0.2
nL=0.8
beta_HH_N0 = 10

O autovalor dominante determina a dinâmica da fase exponencial de crescimento (ou decrescimento) da infecção, i.e., $i_H\propto e^{\lambda_1 t}$ e $i_L\propto e^{\lambda_1 t}$, em que $\lambda_1$ é o autovalor dominante. Isso é evidenciado quando comparamos o coeficiente angular da reta de crescimento exponencial com o autovalor dominante. Logo, se $\lambda_1>0\Rightarrow \mathcal{R}_0>1$ e a doença persiste na população. Por outro lado, se $\lambda_1<0\Rightarrow \mathcal{R}_0<1$ e a doença extingue.

Assumindo que $i_H+i_L=1$, a solução do sistema fornece $\bar{x}=(i_H, i_L)=(0,\!9375, 0,\!0624)$.

A proporção de infectados nos dois grupos na fase de crescimento pode então ser utilizada para estimar o $\mathcal{R}_0$ (soma ponderada)

$$\mathcal{R}_0=\mathcal{R}_0^H\times i_H + \mathcal{R}_0^L\times i_L = 2,08(0,9375)+0,82(0,0624) = 2,01$$

Isso mostra que $0,82\leq 2,01 \leq 2,08$.

Como mostrado nos resultados oriundos do algoritmo, o autovetor associado ao autovalor dominante embora seja parecido, não é o mesmo que o obtido quando utilizamos $i_H+i_L=1$. Neste caso, como calculado acima, o autovetor associado ao autovalor dominante $\lambda_1=1,0013$ é dado por $\bar{v}=(0,\!9977, 0,\!0664)$.
